Serial Killers API

Explore criminal history data with ease

Base URL

https://serial-killers-api.netlify.app/api

Endpoints

Serial Killers Endpoints

Get All Serial Killers

GET /serialkillers

Returns a comprehensive list of all serial killers in the database.

Example Response:

[
    {
        "id": 1,
        "name": "Ted Bundy",
        "active_years": "1974-1978",
        "victims": 30,
        "country": "United States",
        "captured": true
    },
    {
        "id": 2,
        "name": "Jeffrey Dahmer",
        "active_years": "1978-1991",
        "victims": 17,
        "country": "United States",
        "captured": true
    }
    // ... more serial killers
]

Search Serial Killers by Name

GET /serialkillers/name/:name

Perform a flexible search with partial, case-insensitive name matching.

Example Request:

https://serial-killers-api.netlify.app/api/serialkillers/name/bundy

Example Response:

[
    {
        "id": 1,
        "name": "Ted Bundy",
        "active_years": "1974-1978",
        "victims": 30,
        "country": "United States",
        "captured": true,
        "additional_details": {
            "birth_date": "November 24, 1946",
            "death_date": "January 24, 1989",
            "method": "Executed by electric chair",
            "notable_locations": ["Washington", "Utah", "Florida"]
        }
    }
]

Response Object Details

Serial Killer Object Structure

Field Type Description
id Number Unique identifier for the serial killer
name String Full name of the serial killer
active_years String Period during which the serial killer was active
victims Number Confirmed number of victims
country String Country where the serial killer operated
captured Boolean Whether the serial killer was captured
