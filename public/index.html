Search Serial Killers by Name

GET /serialkillers/name/:name

Perform a flexible search with partial, case-insensitive name matching.

Example Request:

https://serial-killers-api.netlify.app/api/serialkillers/name/bundy

Response Object Details

Serial Killer Object Structure

Field Type Description
name String Full name of the serial killer
alias String Any known aliases of the serial killer
birth_date Date Date of birth of the serial killer
death_date Date Date of death, if applicable
nationality String Nationality of the serial killer
gender String Gender of the serial killer
race String Race or ethnicity of the serial killer
active_years String Period during which the serial killer was active
victims_count Number Confirmed number of victims
suspected_victims_count String Suspected number of victims, if any
modus_operandi String The method(s) used by the killer to carry out the crimes
active_countries String Country or countries where the serial killer operated
capture_date Date Date when the serial killer was captured
sentencing String Details about the sentence given to the serial killer
execution_date Date Date of execution if applicable
cause_of_death String Cause of death (e.g., execution, natural causes, etc.)
